## Why
Moinhos Do Dao - Eco Quinta needs a website to showcase their farm activities, provide contact information, and enable cottage bookings for "Boa Vista". The site will serve as the primary digital presence for the eco-farm, allowing visitors to discover activities, view the rental cottage, and get in touch.

## What Changes
- Add landing page with hero section and feature highlight boxes
- Add activities blog with individual pages per activity (text + images)
- Add contact page with email form, physical address, and map location
- Add Boa Vista cottage rental page with photo gallery and booking form
- Create default layout with navigation header and footer
- Add reusable components (HeroSection, HighlightCard, ImageGallery, BookingForm, ContactForm, LocationMap)

## Impact
- Affected specs: farm-website (new capability)
- Affected code: app/layouts/, app/pages/, app/components/, content/activities/
- No breaking changes (greenfield development)
